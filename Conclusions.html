Challenges:
The main challenges faced on this project were time to annotate images and computational speed. The training model ran overnight on a 2012 Macbook Pro with 16 GB 1600 MHz DDR3 of memory and a 2.6 GHz Intel Core i7 processor. Future Work:
To improve the capabilities of this project in the future, it would be advisable to distribute training to a cluster and train on a larger number of annotated images.
